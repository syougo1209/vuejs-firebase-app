# 楽天レシピを用いたレシピ検索アプリ

## アプリ概要
文字入力または音声入力で材料からレシピを検索できるアプリ.気に入ったレシピのお気に入り登録も可能

## 技術一覧
- 楽天レシピ系API
- HTML5
- CSS3
- Vue.js[2.6.11]
- Vue Router
- Vuex
- JavaScript
- Firebase Realtime Database
- Firebase Cloud Firestore
- Firebase Authentication

## 機能一覧
- ユーザー登録
- ログイン
- ログアウト
- レシピの検索(文字、音声)
- レシピのお気に入り登録
